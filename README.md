# AccuKnox AI/ML Assessment

This project contains my solutions for the technical assessment completed as part of the AI/ML position application at AccuKnox.

## What's Included

- **Problem Statement 1 Solutions**: Three Python scripts addressing different aspects of the assessment
- **Database Setup**: SQLite database with sample data
- **Dependencies**: All required packages listed in requirements.txt

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the solution scripts in the `problem_statement1/` directory
